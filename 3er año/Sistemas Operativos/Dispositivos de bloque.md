Los dispositivos de bloque permiten la transferencia de datos en bloques de tamaño fijo (normalemente de varios bytes). Este tipo de dispositivo es adecuado para hardware que requiere el acceso a grandes cantidades de datos de forma eficiente, como los sistemas de almacenamiento.

Ejemplos de dispositivos de bloque:
- Discos duros
- SSD
- Memorias USB
En el sistema, estos dispositivos se encuentran en /dev, con nombres como /dev/sda para un disco duro o una partición específica.
