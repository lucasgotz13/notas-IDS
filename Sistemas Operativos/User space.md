
> [!Info] User-land
> Espacio donde viven las aplicaciones de usuario, a estas se la denominan **procesos**.

Cada proceso o programa en ejecución posee memoria con las instrucciones, los datos, el stack y el heap.

![[Pasted image 20260318172807.png]]

Los procesos se ejecutan en un **contexto**:
- **Aislado**
- **Protegido**
- **Restringido**
Mediante el uso de funciones que se ejecutan en bibliotecas pueden utilizar los servicios de acceso al hardware o **recursos que el kernel proporciona**.
